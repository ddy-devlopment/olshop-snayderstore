Pusat Bantuan Digital Creative

Kami siap membantu Anda dalam menggunakan produk digital dan menyelesaikan masalah yang mungkin Anda hadapi.

Panduan Pengguna

Cara Membuat Akun

Ikuti langkah-langkah berikut untuk membuat akun di Digital Creative:

  1. Klik tombol "Daftar" di pojok kanan atas halaman
  2. Isi formulir pendaftaran dengan data yang valid
  3. Verifikasi email Anda melalui link yang kami kirim
  4. Login dengan akun yang telah dibuat

Cara Membeli Produk

Panduan lengkap untuk membeli produk digital:

  1. Pilih produk yang ingin dibeli dari katalog
  2. Klik tombol "Beli Sekarang" atau "Tambah ke Keranjang"
  3. Lakukan pembayaran sesuai metode yang tersedia
  4. Tunggu konfirmasi pembayaran (biasanya otomatis)
  5. Unduh produk dari halaman "Pesanan Saya"

Cara Mengunduh Produk

Setelah pembayaran berhasil, ikuti langkah-langkah ini:

  1. Login ke akun Anda
  2. Buka halaman "Pesanan Saya"
  3. Cari produk yang telah dibeli
  4. Klik tombol "Unduh" untuk mengakses file
  5. Simpan file di perangkat Anda

Cara Menggunakan Config/Lisensi

Panduan penggunaan config dan lisensi produk:

  1. Buka file config/lisensi yang telah diunduh
  2. Ikuti petunjuk instalasi yang tersedia di file README
  3. Untuk config VPN, impor file ke aplikasi yang sesuai
  4. Untuk lisensi software, masukkan kode saat diminta
  5. Jika mengalami masalah, hubungi dukungan teknis

Cara Bergabung dengan Komunitas

Bergabunglah dengan komunitas Digital Creative:

  1. Kunjungi halaman "Komunitas" di menu utama
  2. Pilih platform yang ingin Anda ikuti (Discord/Telegram)
  3. Klik link undangan yang tersedia
  4. Ikuti aturan dan panduan komunitas
  5. Perkenalkan diri Anda di channel yang disediakan

Pertanyaan Umum (FAQ)

  • Apakah produk bisa direfund?
    Kami menyediakan refund dalam waktu 24 jam setelah pembelian jika produk tidak sesuai deskripsi atau mengalami masalah teknis yang tidak dapat diselesaikan. Produk digital yang telah diunduh tidak dapat direfund kecuali ada masalah yang tidak dapat diperbaiki.
  • Berapa lama link download aktif?
    Link download produk aktif selama 30 hari setelah pembelian. Setelah itu, Anda masih dapat mengunduh ulang dengan menghubungi dukungan pelanggan. Kami sarankan untuk menyimpan file di tempat yang aman setelah mengunduh.
  • Bagaimana cara mendapatkan dukungan teknis?
    Anda dapat menghubungi dukungan teknis melalui email support@digitalcreative.id, WhatsApp di +62 812-3456-7890, atau melalui grup komunitas Discord/Telegram. Sertakan detail pesanan dan deskripsi masalah yang Anda hadapi.
  • Apakah pembayaran diverifikasi otomatis?
    Ya, sebagian besar pembayaran melalui transfer bank, e-wallet, dan QRIS diverifikasi secara otomatis dalam waktu 1-5 menit. Untuk pembayaran dengan metode tertentu, verifikasi mungkin membutuhkan waktu hingga 1x24 jam.
  • Bagaimana jika file rusak atau tidak bisa dibuka?
    Jika file rusak atau tidak dapat dibuka, silakan hubungi dukungan pelanggan dengan menyertakan bukti pembelian dan screenshot masalah. Kami akan mengirim ulang file atau memberikan solusi alternatif dalam waktu 1x24 jam.

Masalah Teknis Umum

Produk Tidak Muncul

Jika produk tidak muncul setelah pembayaran:

  1. Refresh halaman "Pesanan Saya"
  2. Periksa email konfirmasi pembayaran
  3. Pastikan Anda login dengan akun yang benar
  4. Hubungi dukungan jika masalah berlanjut

Gagal Mengunduh File

Solusi jika mengalami masalah saat mengunduh:

  1. Periksa koneksi internet Anda
  2. Coba gunakan browser berbeda
  3. Nonaktifkan ekstensi/ad-blocker sementara
  4. Coba unduh di perangkat lain
  5. Hubungi dukungan jika tetap gagal

API Error atau Koneksi Bermasalah

Jika mengalami masalah koneksi atau API error:

  1. Periksa status server di halaman status kami
  2. Coba akses setelah beberapa menit
  3. Clear cache dan cookies browser
  4. Gunakan koneksi internet yang stabil
  5. Laporkan masalah jika berlanjut

Config atau Server Tidak Berfungsi

Jika config atau server tidak berfungsi:

  1. Periksa panduan penggunaan config
  2. Pastikan aplikasi client terbaru
  3. Coba config di perangkat/network berbeda
  4. Periksa status server di halaman status
  5. Hubungi dukungan teknis untuk bantuan

Kebijakan & Informasi

Kebijakan Refund

Kebijakan refund kami:

  • Refund diberikan dalam 24 jam jika produk tidak sesuai deskripsi
  • Produk digital yang telah diunduh tidak dapat direfund
  • Refund tidak berlaku untuk masalah yang disebabkan oleh kesalahan pengguna
  • Proses refund membutuhkan waktu 3-5 hari kerja

Hak Cipta & Lisensi

Informasi hak cipta dan lisensi:

  • Semua produk dilindungi hak cipta
  • Lisensi berlaku untuk penggunaan pribadi
  • Dilarang memperjualbelikan ulang produk
  • Dilarang membagikan produk kepada pihak lain

Ketentuan Penggunaan

Ketentuan penggunaan platform:

  • Pengguna bertanggung jawab atas keamanan akun
  • Dilarang menggunakan platform untuk aktivitas ilegal
  • Kami berhak menangguhkan akun yang melanggar ketentuan
  • Ketentuan dapat berubah sewaktu-waktu

Privasi & Keamanan Data

Kebijakan privasi dan keamanan data:

  • Data pribadi dilindungi dan tidak dibagikan
  • Pembayaran diproses dengan enkripsi SSL
  • Kami tidak menyimpan data kartu kredit
  • Pengguna dapat menghapus data pribadi kapan saja

Tips & Rekomendasi

Tips Penggunaan Produk Digital

Beberapa tips untuk memaksimalkan penggunaan produk digital:

  • Selalu baca dokumentasi sebelum menggunakan produk
  • Backup file penting sebelum melakukan modifikasi
  • Gunakan perangkat lunak terbaru untuk kompatibilitas optimal
  • Bergabunglah dengan komunitas untuk mendapatkan bantuan
  • Periksa update produk secara berkala untuk fitur terbaru
  • Gunakan antivirus untuk memindai file sebelum membuka
